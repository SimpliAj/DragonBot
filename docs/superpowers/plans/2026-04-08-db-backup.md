# DB Backup System Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Automatically back up `dragon_bot.db` to `backups/` every 12 hours, and let the developer trigger a manual backup via a button in `/devpanel`.

**Architecture:** New `cogs/backup.py` contains a standalone `backup_db()` async function (using SQLite's `.backup()` API for a safe hot-copy) and a `BackupCog` with a `@tasks.loop(hours=12)`. The devpanel's `DangerView` gets a "💾 Backup DB" button that calls `backup_db()`. `main.py` loads the new cog.

**Tech Stack:** Python 3.11+, discord.py (`discord.ext.tasks`), `sqlite3` (stdlib), `os` / `datetime` (stdlib)

### Task 1: Create `cogs/backup.py`

**Files:**
- Create: `cogs/backup.py`

This file owns everything backup-related: the `backup_db()` async function and the `BackupCog` with its 12-hour loop. Follow the exact same `@tasks.loop` + `before_loop` pattern used in `cogs/tasks.py`.

The database path comes from `config.DB_PATH` (currently `'dragon_bot.db'`). Backups go into a `backups/` subdirectory (relative to where the bot is run from). The filename includes a UTC timestamp so every backup is unique.

SQLite's `sqlite3.Connection.backup(dest)` performs an online, WAL-safe hot backup — it is safe to call while other connections are reading/writing.

- [ ] **Step 1: Create the file with `backup_db()` and `BackupCog`**

  Create `cogs/backup.py` with the following complete content:

  ```python
  """
  cogs/backup.py - Automatic and manual SQLite database backups.
  """

  import asyncio
  import logging
  import os
  import sqlite3
  from datetime import datetime, timezone

  import discord
  from discord.ext import commands, tasks

  from config import DB_PATH

  logger = logging.getLogger(__name__)

  BACKUP_DIR = 'backups'


  async def backup_db() -> str:
      """
      Create a hot backup of DB_PATH into the backups/ directory.

      Uses sqlite3.Connection.backup() — safe with WAL mode and concurrent
      connections. Returns the backup filename. Raises on failure.
      """
      os.makedirs(BACKUP_DIR, exist_ok=True)
      timestamp = datetime.now(timezone.utc).strftime('%Y-%m-%d_%H-%M-%S')
      dest_path = os.path.join(BACKUP_DIR, f'dragon_bot_{timestamp}.db')

      def _do_backup():
          src = sqlite3.connect(DB_PATH, timeout=30.0)
          dst = sqlite3.connect(dest_path, timeout=30.0)
          try:
              src.backup(dst)
          finally:
              dst.close()
              src.close()

      await asyncio.get_event_loop().run_in_executor(None, _do_backup)
      logger.info(f'Database backed up to {dest_path}')
      return dest_path


  class BackupCog(commands.Cog):
      def __init__(self, bot):
          self.bot = bot
          self.scheduled_backup.start()

      def cog_unload(self):
          self.scheduled_backup.cancel()

      @tasks.loop(hours=12)
      async def scheduled_backup(self):
          try:
              path = await backup_db()
              logger.info(f'Scheduled backup complete: {path}')
          except Exception as e:
              logger.error(f'Scheduled backup failed: {e}')

      @scheduled_backup.before_loop
      async def before_scheduled_backup(self):
          await self.bot.wait_until_ready()


  async def setup(bot):
      await bot.add_cog(BackupCog(bot))
  ```
